Setup and running of the framework:

1 Install nodejs and npm on your meachine

2 Install Webdriverio for the project specific in command prompt by  $ npm install webdriverio --save-dev

3. Start Selenium Server by $  selenium-standalone start

4. To configure the WIDO runner , from the project cmd run "wdio config" and follow the instructions.

5. Install Mocha by  npm install wdio-mocha-framework --save-dev
